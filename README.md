# Chapter-5
第五章算例中涉及的数据与产消者参数

# 概述
本项目是博士论文 《智能配电网产消者能量共享优化方法研究》-“第五章 面向柔性互联社区的完全分布式优化方法” 中案例研究参数的说明文档。

> _**16-node case**_：16 节点下产消者的能量共享案例研究参数。  
> _**126-node case**_：126节点下产消者的能量共享案例研究参数。   

每个文件夹中含有资源参数以及负荷两类数据。

# 1. 资源参数文件 (ResourceParameter.xlsx)
该文件以多工作表的形式，精细划分了多社区产消者中不同类型资产的运行特征。各工作表具体含义如下：

* **`TG`**
  * 记录了传统发电机组的运行参数，通常包含机组的最大/最小出力限制以及发电成本系数（二次项、一次项和常数项）等。
* **`RES`**
  * 记录了可再生能源资源在各个时段的理论发电上限。
* **`ESS`**
  * 记录了储能设备的物理约束，包括最大充放电功率限制、充放电效率等。
* **`SOP`**
  * 专门记录柔性互联装置的参数，包括其额定容量及其链接节点与社区。
* **`FL`**
  * 记录了灵活负荷的相关参数。
* **`Price`**
  * 记录了多社区与上层电网交互的购电价（buy）与售电价（sell）。

# 2. 节点负荷文件 (NodeLoad.xlsx)
该文件用于描述各节点产消者的基础负荷需求。
